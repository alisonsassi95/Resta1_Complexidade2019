# Trabalho Complexidade, Jogo Resta 1

Desenvolver um programa para jogar RESTA 1, seguindo as especificações gerais abaixo:

* Ler a posição onde existe o primeiro espaco vago inicial.

* Ler a posição onde restará o último "pino".

* Mostrar a lista de possíbilidade(s)para chegar a solução final. OU a sua impossibilidade.

Trabalho pode ser feita em dupla.

Nosso Trabalho tem a possibilidade de jogar manualmente, interagindo com a interface gráfica.
